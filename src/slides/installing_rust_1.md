### Installing Rust

An official Rust tool, [rustup](https://rustup.rs), can be used to install Rust and help you manage the versions and update.
<pre>
    <code class="shell hljs">
        $ curl https://sh.rustup.rs -sSf | sh
        $ . $HOME/.cargo/env
        $ rustup install nightly
        $ rustup update
    </code>
</pre>
